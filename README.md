# Tony Skyview — 드론 영상 소개 페이지

## 프로젝트 소개

개인 드론으로 찍은 제주 주변 사진을 모아 두고, **넷플릭스 메인 화면처럼 어둡고 카드형**으로 보여 주는 연습용 웹 페이지입니다.  
실제 영상 재생 기능은 없고, **HTML과 CSS만**으로 레이아웃과 분위기를 맞춰 본 프로젝트입니다.

## 사용 기술

- HTML5 (`header`, `main`, `section`, `footer`, `article` 등 시맨틱 태그)
- CSS3 (플렉스, 호버 효과, 그라데이션 등)
- JavaScript는 사용하지 않습니다.

## 구현한 화면 구성

| 영역 | 내용 |
|------|------|
| **헤더** | 사이트 이름(TONY SKYVIEW), 짧은 부제 |
| **배너(히어로)** | 송악산 대표 이미지(`songak.png`)와 제목·설명 문구 |
| **추천 영상 줄** | 드론 썸네일 카드 5개(해식동굴, 대포주상절리대, 성산일출봉, 범섬 해식동굴, 갯깍주상절리대) |
| **푸터** | 저작 표시 문구 |

이미지 파일은 `images` 폴더에 두고, HTML에서 경로로 불러옵니다.

## 작업하면서 수정한 내용

처음에는 배너에 **배경 이미지(`background-image`)**를 쓰고 `cover`로 채웠더니, 송악산 사진 **위·아래가 잘리는** 문제가 있었습니다.  
`contain`으로 바꾸고 높이를 제한하는 방식도 시도했는데, 박스 높이와 `overflow` 때문에 여전히 잘릴 수 있어서, 결국 **`<img>` 태그**로 바꾸고 `width: 100%`, `height: auto`로 두어 **이미지 비율 그대로** 배너 높이가 잡히도록 정리했습니다.

카드는 처음에 4개였다가 `tafoni.jpg`를 추가해 **5개**로 늘리고, 각 장소에 맞게 **제목·설명 문구**를 표에 맞춰 수정했습니다.  
로고 문구는 **SKYVIEW → TONY SKYVIEW**, 푸터는 과제·연습용 표기로 **영문 한 줄**로 바꿨습니다.

## AI를 활용한 방식

원하는 화면 느낌과 제약(HTML/CSS만, 구조 단순하게 등)을 **짧은 프롬프트**로 여러 번 나눠 요청했습니다.  
첫 버전을 브라우저로 본 뒤, 잘리는 부분이나 카드 개수처럼 **눈에 보이는 문제**를 구체적으로 말하면서 수정을 이어 갔습니다.  
코드가 길어지기 전에 구조를 고정해 두고, 나중에는 **텍스트만 고치는** 식으로 범위를 줄여 작업했습니다.

## 실행 방법

1. 이 저장소(폴더)를 그대로 둡니다.
2. **`index.html`** 파일을 더블클릭하거나, 브라우저(Chrome 등)에서 **파일 열기**로 연다.
3. `images` 폴더와 `styles.css`가 `index.html`과 같은 위치에 있어야 그림과 스타일이 정상 적용됩니다.

별도 서버나 빌드 과정은 없습니다.

## 느낀 점

배너처럼 **“꽉 차 보이게”** 하려다 보면 이미지가 잘리기 쉽다는 걸 직접 겪었습니다.  
목표가 “화면을 꽉 채우기”인지 “사진 전체 보이기”인지에 따라 HTML/CSS 선택이 달라진다는 것도 알게 되었습니다.  
시맨틱 태그로 영역을 나눠 두니, 나중에 글자만 바꿀 때도 찾기 편했습니다.
